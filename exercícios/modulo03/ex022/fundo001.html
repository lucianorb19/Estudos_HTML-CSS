Exemplo com 3 divs, todas pertencentes a classe quadrado, cada uma com um id diferente.

Na primeira div, uma cor padrão de fundo. Na segunda, um gradiente, e na terceira, uma imagem, todas configuradas pela id na CSS.

background-color


        background-color: lightblue;

background-image: linear-gradient

           
        background-image: linear-gradient(to top, red,yellow);

background-image: url()

  
        background-image: url('imagens/pattern003.png');

Para o fundo do body, também pode ser definida uma imagem, como está sendo mostrado, tanto usando uma imagem local, quanto uma url externa.
