Tabelas Simples

A criação de tabelas simples é feita com a tag table. E dento dessa tag, são usadas outras tags, seguindo a hierarquia:

  • tr - table row - linha
  • th - table header - cabeçalho (usar para títulos da tabela)
  • td - table data - dado da célula

Dica: Fechamento opcional das tags tr e td.

Dica: Dados fora de ordem ou fora da hieraquia, aparecem antes ou depois da tabela.

Para criar as bordas típicas de uma tabela, usa-se, na css: 


    table{
        border-collapse: collapse; /*Não criar divisões entre as células*/
    }

    td{ /*Personalização para cada célula*/
        border: 1px solid black;
        padding: 5px;
    }

Scope

Para o parâmetro th, usado para títulos de coluna ou linha, quando for título de uma coluna, o parâmetro scope assume valor col, quando for título de uma linha, assume valor row. Isso ajuda os mecanismos de busca a valorizar o site.

No exemplo abaixo, Nomes é um título (th) com escopo col

Nomes
Luciano
Abreu
Daniel

Nesse exemplo, o título Total tem escolpo row

Nome Valor (R$)
Banana 15.10
Maçã 12.50
Total 27.60

Quando o título estiver mesclado e se referir a um conjunto de linhas ou colunas, ele vai ser colgroup ou rowgroup

Alinhamento Horizontal e Vertical

Alinhamento horizontal com parâmetro text-align. Alinhamento vertical com parâmetro vertical-align, assumindo valores top middle e bottom.

Tabelas Grandes

Tabela maiores podem ser divididas em:

  • caption - uma pequena frase que resume a tabela
  • thead - Table Head - Cabeçalho da tabela
  • tbody - Tabel Body - Corpo da tabela
  • tfoot - Table Foot - Pé da tabela
Cada uma dessas estruturas pode conter tr, th e td.

Dica: Usar a estrutura thead, tbody e tfoot para qualquer tabela, não somente as maiores.

Efeito Zebrado

Para usar o efeito zebrado na tabela, utilizar a pseudoclasse nth-child(2n-1). Ela seleciona linhas em intervalos (a cada duas, a cada três....). Na CSS, aplicando somente ao body da tabela:


    tbody > tr:nth-child(2n){/*Para o corpo da tabela, para cada linha, pulando uma*/
        background-color:lightgray;
    }

Também pode ser usado, no lugar do 2n, o valor odd, que delimita as linhas pares, ou even, que delimita as ímpares.

Cabeçalho Fixo

Para manter o cabeçalho fixo na tabela, independente da rolagem para baixo, usam-se duas configurações. No seletor table, definir position: relative; e criar um outro seletor para o cabeçalho, da seguinte forma: 


    table{
        position: relative;
    }

    thead > tr > th{/*Para os títulos filhos de tr(linhas) dentro do cabeçalho da tabela*/
        position:sticky; /*Posição gruda*/
        top: 0;/*na parte de cima da tela */
        background-color: gray;/*mesmo do cabeçalho da table*/
        }

Mesclagem de Células

A mesclagem de células é feito usando os parâmetros colspan e rowspan diretamente nas células. colspan é usado para mesclar as células em colunas e rowspan em linhas.

Por exemplo, para que a célula B ocupe o espalo de duas células, e para que a célula D ocupe três linhas


    colspan="2"
    rowspan="3"
A B
D E F
H I
K L

Operações em Colunas

Para realizar alguma mudança nas linhas, é necessário somente usar a tag tr. Para colunas, como não há uma tag específica na criação da tabela, o HTML5 trouxe uma nova tag, colgroup, e dentro dessa tag, é possível criar elementos col com classes, sendo cada um uma coluna da tabela, sendo o primeiro col a coluna mais a esquerda, e assim por diante. Por exemplo, a tabela:

Nome Sexo Idade
Luciano M 27
Gustavo M 24
Lucas M 28

Usando a tag colgroup:


    colgroup 
        col class="col_nome";
        col class="col_sexo";
        col class="col_idade"
E na CSS, configurar algo específico para cada class da col 

    col.col_nome{
        background-color: gray;
    }
Nome Sexo Idade
Luciano M 25
Gustavo M 24
Lucas M 28

Tambem pode ser usado o parâmetro span dentro da tag col, para que a configuração da coluna se expanda para X colunas a direita dela.


    col class="col_nome" span="2"

Isso significa que contando a coluna col_nome, duas colunas terão a configuração dessa classe.

Responsividade com Tabelas

Para criar tabelas que sejam estéticas e responsivas, uma opção é envelopar toda a tag table com uma div de id qualquer, e nesse id, usar os parâmetros overflow.

  • Overflow: responsividade em todos os eixos
  • Overflow-x: responsividade no eixo x
  • Overfow-y: responsividade no eixo y

Os valores para o parâmetro overflow podem ser auto, hidden e scroll. Assumindo valor auto, o navegador cria uma barra de rolagem somente para a tabela. Ainda pode ser usado o parâmetro width assumindo valor 50vw, ou seja a tabela, com a barra de rolagem, vai sempre ocupar 50% da largura da tela.
