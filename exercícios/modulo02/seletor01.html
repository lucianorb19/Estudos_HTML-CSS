Usando ID 3

O ID serve pra diferenciar um seletor dos demais. Assim, pode ser aplicada uma configuração CSS personalizada somente para este seletor.

Para criar um seletor com id, colocar como parâmetro da tag de abertura dele id="nome", com nome sendo o que você escolher.

E para configurá-lo no CSS, usar #nome{}, passando as configurações desejadas.

Dica: Em HTML é id, em CSS é # / em html é class, em CSS é .

Dica: Só se deve usar um id por seletor na página HTML. Colocando o mesmo id para vários seletores, funciona, mas não é a recomendação do W3E. Nesse caso, usaria-se a class

Usando Class

A class é usada para aplicar a mesma configurações a vários seletores, quando isso for pertinente. Para criar seletores com class, usar na tag de abertura class="nome", e usar .nome{} para configurar na CSS

Usando Class 2

Dica: Podem ser usadas mais de uma classe no seletor - class="nome nome2". E também podem ser usados class e id no seletor - id="nome" class="nome".

Dica: tag span pode ser usada somente para selecionar algum trecho de texto a aplicar id/class a ele

Dica: Id sobrepõe class

  • ID - html id='nome' - CSS #
  • Class - html class='' - CSS .nome{}
