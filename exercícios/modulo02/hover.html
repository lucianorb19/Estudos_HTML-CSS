Pseudo-Classe com Parágrafo Filho da Div

Dentro desse conceito, tem como criar um parágrafo dentro da div e fazer algo específio nele com as pseudoclasses.

Primeiro, para definir que algo vai ser feito dentro das tags p que estão dentro de uma div, usar na CSS:


    div > p{/*para os parágrafos filhos de div (que estão dentro da div)*/
            display:none; /*Não mostrar na tela*/
    }

Dica: div > p se refere a todas as tags p diretamente dentro de tag div. Para referenciar qualquer tag p (direta e indiretamente) dentro de div, usa-se div p

Logo, todos parágrafos dentro de div não serão mostrados na tela. Para fazer com que apareçam, cria-se uma pseudoclasse div:hover >p{} na CSS, que é uma pseudo-classe para todos os parágrafos filhos de div, onde se passa o mouse em cima.


    div:hover > p{
        display: block;
        width: 200px;
    }
