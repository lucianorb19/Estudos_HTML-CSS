Pseudo-classes e Pseudo-elementos

Personalizando links com Pseudo-classes visited e active

Links podem ser personalizados com as pseudo-classes visited e active.
visited - se o site já foi visitado recentemente no dispositivo;
active - se o site está sendo clicado.

Pseudo-Elementos

São elementos que modificam o conteúdo de elementos, classes, id,... . Para configurar um pseudo-elemento na CSS, usar x::y, sendo x o seletor a ser modificado e y o pseudo-elemento.

Por exemplo, todas as vezes que tiver um link, mostrar a frente dele a palavra [link]:


    a::after{
        content: ' [link]'
    }
  • ID - html id='nome' - CSS #
  • Class - html class='' - CSS .nome{}
  • Pseudo-Class - Na CSS seletor:ação{} (div:hover{})
  • Pseudo-Elemento - Na CSS seletor::posição (a::after{content: ' [link]'})
