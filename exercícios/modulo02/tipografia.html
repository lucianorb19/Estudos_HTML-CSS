Tipografia

Glifo - Letra
Conjunto de glifos de uma determinada família - Fonte
Família Tipográfica - Letras de uma mesma fonte, mas com variações. Cada uma dessas variações pertence a uma família tipográfica Categoria - uma característica da fonte (fonte serifada é uma categoria, fonte sans-serif (sem serifa) é uma categoria, monoespaçada,hadwriting,display,...)

Web Safe Font Combination

É uma técnica que se baseia em declarar, na CSS, várias fonts para o parâmetro font-family, porque caso o dispositivo não possua uma das fontes, ele tenta usar a fonte seguinte, que possui propriedades parecidas.

Nesse mesmo contexto, ao invés de usar uma fonte, também pode ser usado um valor de categoria. Nesse caso, o próprio dispositivo escolhe a fonte pela categoria.

Tamanhos de Fontes

Configurada na tag font-size, as medidas mais recomendadas são px e em. Por padrão, o tamanho da fonte nos navegadores é 16px.
16px = 1em

Medidas Absolutas

cm, mm, in (inche-polegada), px, pt, pc (paica? )

Medidas Relativas

  • em - relativa ao tamanho da altura M maiúsculo da fonte
  • ex - relativa à altura x da fonte
  • rem - relativa à altura M maiúsculo configurada no root
  • vw (view-width) - relativa à largura da tela
  • vh (view-hight) - relativa à altura da tela
  • % - porcentagem da tela

Peso da Fonte

O quanto de negrito é aplicado à fonte, usando à tag font-weight. As variações são lighter / normal / bold / bolder ou de 100 a 900. Importante ressaltar que a maioria das fontes não aceita todas essas variações.

Font Style

Tag que permite mudar o estilo da fonte. Suas variações são italic / normal / oblique / initial /.... Nem todas as fontes tem essas variações.

Text-decoration

Tag que assume valor underline e deixa o texto sublinhado.

ShortHand

Técnica de simplificar a declaração dos parâmetros da fonte.É preciso escrever na ordem
font-style-> font-weight-> font-size-> font-family.
E caso não haja algum parâmetro, só é preciso omití-lo e continuar seguindo a ordem.
Ao invés de escrever

    
    h1 {
        font-family: 'Work Sans', sans-serif;
        font-weight: bolder;
        font-size: 3em;
        font-style: italic;
    }
se escreve: 

    font: italic bolder 3em 'Work Sans', sans-serif;
