Pseudo-Classes

Pseudo-classes são classes específicas para determinados estados, criadas na CSS. Por exemplo, uma pseudo-classe hover, que faz com que, o seletor ligado a ela faça algo específico quando o mouse passa em cima do seu seletor.

Dica: Transition duration: Xs; - atrasa o efeito do hover em X segundos.

  • ID - html id='nome' - CSS #
  • Class - html class='' - CSS .nome{}
  • Pseudo-Class - Na CSS seletor:ação{} (div:hover{})
  • Pseudo-Elemento - Na CSS seletor::posição (a::after{content: ' [link]'})
