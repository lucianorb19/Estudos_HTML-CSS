Introdução

Grouping Tags são tags que dividem o corpo do site em header (cabeçalho), main (conteúdo principal) e footer(rodapé).

Dentro destes, podem ser inseridas outras tags para organizar o conteúdo - article, section, aside,... . E para personalizar, para cada grouping tag, cria-se uma personalização diferente no CSS

Sombras

Tag box-shadow com 5 parâmetros: 1º - horizontal, 2º Vertical, 3º Blur, 4º Espalhamento, 5º Cor.

Antes desses valores, é possível definir o modo inset, no qual a sombra aparece no interior da imagem, ao invés da margem.

Sempre usar com transparência e quase imperceptível.

Bordas

Arredondamento da borda usando border-radius. Se forem passados 4 parâmetros (em cima esquerda, em cima direita, embaixo direita, embaixo esquerda). Um único parâmetro é para todos. Dois parâmetros, primeiro é em cima esquerda e embaixo direita.

Dica: border-radius: 50% - bordas redondas

Bordas compostas por imagens - usando as tags
border-image-source: url(caminho arquivo);
border-image-slice: 10px; border-imagem-repet: repeat;

No shorthand:
border-image: url(caminho arquivo) 10px repeat

Para que funcione corretamente, usar imagens de tamanho 80px x 80px e baixa resolução (70x70).
