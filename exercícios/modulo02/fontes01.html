Fontes

Google Fontes

Para utilizar fontes do Google Fontes, é preciso acessar o site, escolher a fonte, selecionar quais variações você deseja, e selecionar @import. Copiar o código resultando e adicionar à CSS. O código CSS é uma regra que importa a fonta toda vez que a página é acessada.

Download de Fontes

Também é possível fazer o download do arquivo da fonte (formato otf, ttf). Basta acessar algum site que ofereça esse serviço, como o dafonte.com. Se atentar para usar uma fonte que tenha glifos acentuados e minúsculo/maiúsculo, caso seja necessário.

Após o download, colocar os arquivos na pasta do projeto e usar dentro do style a tag/regra:


    @font-face{
        font-family: ;
        src: url() format(),
             url() format();
    }

O valor para font-family é um nome que você quiser. Em url é o caminho do arquivo fonte, e em format é um valor que depende do formato do arquivo fonte e do navegador

  • opentype - (otf)
  • truetype - (ttf)
  • embedded-opentype
  • truetype-aat (Apple Advanced Typography)
  • svg

Tendo tudo isso configurado, só é preciso utilizar a font-family de nome escolhido na configuração CSS dos seletores.

No exemplo há dois url e dois format porque podem ser configuradas duas fontes/formatos de fonte ao mesmo tempo (ou mais).

Captura de Fontes

Extensão Fonts Ninja do Chrome - possibilita identificar a font-family de qualquer texto na tela. Tendo o nome, é possível baixar a fonte ou usá-la embedada.

Para identificar a fonte dentro de uma imagem podem ser usadas 3 site:

  • whatfontis.com
  • fontsquirrel.com
  • myfonts.com

Alinhamento de Texto

Nos seletores CSS, com o parâmetro text-align, recebendo os valores center, right, left, justify.

Para fazer com que a primeira linha do parágrafo tenha mais recuo, parâmetro text-ident, com valor em px(20 a 30 ou mais). Para definir a distância horizontal entre as linhas, line-height.
