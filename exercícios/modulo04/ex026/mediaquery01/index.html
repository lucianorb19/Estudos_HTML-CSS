Sobre Media Queries

Media Query é um recurso que permite definir diferentes estilos CSS para visualização/impressão da página. São definidos diferentes arquivos CSS, uma para visualiação em tela, outro para impressão. Isso é feito pela definição do arquivo CSS na tag link, usando o parâmetro media, que assume diferentes valores (media types)

  • screen: visualização em tela;
  • print: impressão;
  • all: qualquer formato. É usado para as definições em comum independente da media type.

Tags inteiras podem ser "escondidas" na CSS, usando o parâmetro display:none;
