Media Features

As media features também podem ser definidas localmente, dentro da tag style do próprio arquivo HTML, ou num arquivo CSS único, usando, para cada tipo de orientação uma regra .


    style
    @media screen and (orientation: portrait){
        /*Configurações retrato*/
    }

    @media screen and (orientatiom: landscape){
        /*Configurações paisagem*/
    }

Device BreakPoints

São definições de tamanho para cada tipo de dispositivo.

  • Telas pequenas - até 600px
  • Celular: de 601px a 768px
  • Tablet: de 769px a 992px
  • Desktop: 993px a 1200px
  • Telas grandes: acima de 1200
