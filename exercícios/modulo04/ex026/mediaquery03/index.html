Media Features

Media Features são recursos para especificar ainda mais o que as Media Types fazem. Ao invés de delimitar somente para telas, com o uso de Media Features, pode-se usar um arquivo CSS somente para telas com orientação retrato e outro arquivo somente para telas com orientação paisagem. Isso é feito especificando no parâmetro media da tag link CSS


    link rel="stylesheet" href="estilos/retrato.css" media="screen and (orientation: portrait)"
    link rel="stylesheet" href="estilos/paisagem.css" media="screen and (orientation: landscape)"

Nesse trecho os media features são "(orientation: portrait)" e "(orientation: landscape)". Se houvesse mais de uma media feature por css, seriam todos separados por parênteses, cada um.
