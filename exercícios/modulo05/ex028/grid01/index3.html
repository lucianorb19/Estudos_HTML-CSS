Propriedades para itens

  • grid-row-start
  • grid-row-end
  • grid-column-start
  • grid-column-end

Essas quatro propriedades iniciais são utilizadas para definir uma grid area.

Exemplo


    div.container{
                display:grid;
                border: 1px solid black;
                height: 400px;
                width: 600px;
                grid-template-rows: repeat(3,100px);
                grid-template-columns: repeat(4, 1fr);
                gap: 5px;
            }

            div#caixa1{
                grid-row-start: 1;
                grid-row-end: 2;
                grid-column-start: 1;
                grid-column-end: -1; /*ÚLTIMA COLUNA*/
                background-color: rgba(255, 0, 0, 0.31);
            }

            div#caixa2{
                grid-row-start: 2;
                grid-row-end: 4;
                grid-column-start: 1;
                grid-column-end: 3;
                background-color: rgba(0, 255, 255, 0.29);
            }

            div#caixa3{
                grid-row-start: 2;
                grid-row-end: 4;
                grid-column-start: 3;
                grid-column-end: 5;
                background-color: rgba(172, 255, 47, 0.312);
            }

Shorthands

Para grid-row-start e grid-row-end -> grid-row.
Para grid-column-start e grid-column-end -> grid-column


    grid-row-start: 2;
    grid-row-end: 4;
    grid-column-start: 3;
    grid-column-end: 5;

É o mesmo que:


    grid-row: 2/4;
    grid-column: 3/5;

Há também o shorthand ainda mais simplificado. grid-area.


    grid-area: 2/3/4/5; 
    /*
    2 - grid-row-start
    3 - grid-column-start
    4 - grid-row-end
    5 - grid-column-end
    */
