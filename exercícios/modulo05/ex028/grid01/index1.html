place-items / place-content

Shorthands que simplificam as declarações de align-items e justify-items / align-content e justify-content. Lembrando que: primeiro o align, depois o justify.

Escrever:


        div.container{
            display: grid;
            grid-template-columns: auto auto auto;
            grid-template-rows: 100px 100px 100px;
            align-items: center;
            justify-items: stretch
            align-content: start;
            justify-content: space-around;
        }

É o mesmo que


    div.container{
        display: grid;
        grid-template-columns: auto auto auto;
        grid-template-rows: 100px 100px 100px;
        
        place-items: center stretch;
        place-content: start space-around;
        
    }

grid-template

Shorthand que simplifica a declaração de grid-template-rows e grid-template-columns. Lembrando que: primeiro a linha, depois a coluna.

Escrever:


    grid-template-rows: 100px 100px 100px;
    grid-template-columns: auto auto auto;

É o mesmo que


    grid-template: 100px 100px 100px / auto auto auto;

função repeat()

Serve basicamente para simplificar a declaração de propriedades que são palavras repetidas. Por exemplo


    grid-template-rows: auto auto auto;

Pode ser escrita


    grid-template-rows: repeat(3, auto);

Fraction

Uma maneira de definir o tamanho das áreas de cada item do container usando a medida fr

Exemplo: 


        div.container{
            display:grid;
            grid-template-rows: auto auto auto;
            grid-template-columns: 1fr 1fr 1fr ;
        }

            grid-template-rows: auto auto auto;

Também podem ser usadas medidas mistas na definição. Por, exemplo, um container de 1500px que se divide da seguinte forma:


    div.container{
        display: grid;
        grid-template-rows: auto auto auto;
        grid-template-columns: 100px 20% 1fr 1fr
    }

Destacando que, as medidas de fr usadas representam as frações da área não ocupada da linha, ou seja, no caso acima, 1100px livres divididos em duas colunas de 550px.
