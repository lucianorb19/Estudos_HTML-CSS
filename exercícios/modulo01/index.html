Áudio

A reprodução de áudio no site pode ser feita pela tag audio.
O parâmetro preload pode assumir 3 valores: auto(que só executa após carregar todo o arquivo); metadata (que baixa alguns metadados) e none (que não carrega nada).
autoplay reproduz o áudio automaticamente. controls exibe uma interface simples para o áudio. loop mantém o áudio em loop.
O parâmetro source src é usados para que o código tenha compatibilidade com diversos tipos de arquivo de áudio (mp3,ogg,wav,...). Ele tenta executar primeiro a linha mais acima, e vai descendo a medida que não consegue. Ao final pode ser colocado uma mensagem que acusa a impossibilidade de executar o áudio, com uma opção para download.

Dica: arquivo wav são muito pesados. Procure evitá-los.

            audio preload="metadata" autoplay controls loop>
                source src="midia/dbz.mp3" type="audio/mpeg">
                source src="midia/dbz.ogg" type="audio/ogg">
                source src="midia/dbz.wav" type="audio/wav">
                p>Navegador não reproduz o áudio!   a href="midia/dbz.mp3">Baixar a> p>
            audio>

Vídeos Externos

Existem serviços de hospedagem de vídeo que podem ser usados, já que usar estes serviços é menos custoso (banda) do que hospedar localmente. Nesse caso, só é necessário referenciar no código ao vídeo do serviço externo. No youtube, por exemplo, os vídeos podem ser incorporados, na opção "incorporar", que já gera o código html necessário com a tag iframe. Importante ressaltar que esse serviço do youtube só permite disponiblizar vídeos que são públicos para todos.

Outra opção viável é o vimeo, que ofere o mesmo serviço, com opções pagas, mas com opções de controle de acesso (vídeos privados).

Estilos Inline (prática trabalhosa)

As CSS podem ser aplicadas diretamente nas linhas das tags com os parâmetros desejados. Essa abordagem é trabalhosa e é melhor para ser usada em ajustes pontuais. Por exemplo, em uma tag h2 definir style="color:mediumturquoise;"

Sub-título

Ou definir a cor de fundo com o parâmetro style="background-color:..."

Outro Sub-título

Estilos Locais/Internos

A definição de estilos também pode ser feita na tag head, usando a tag style. Nesse caso, todas as definições para cada seletor (body, h1,h2,p,...) é aplicada a toda a página html atual.
As desvantagens desse método são: o código para o estilo geralmente é maior que o próprio código do corpo da página; para aplicar o estilo, é preciso aplicar o código para cada página html.

Estilos Externos

As definições de estilo podem ser feitas em arquivos CSS separados do HTML. Esses aquivos .css só precisam ser referenciados na head com a tag link:css.
Essa técnica é mais versátil, e permite criar diferentes aquivos CSS com diferentes estilos. Além disso, numa mesma página, podem ser adicionais mais de um CSS, caso haja a necessidade de estilos adicionais.
Lembrando que: a prioridade das configurações CSS é maior no Inline -> Local -> Externo.
